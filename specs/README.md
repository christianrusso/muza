# Specs

Specs de features de LookLab, siguiendo el flujo de Spec-Driven Development descripto en [research-development/engineering-guidelines/02-spec-driven-development.md](../research-development/engineering-guidelines/02-spec-driven-development.md).

Un archivo por feature: `<feature-slug>.md`, usando [TEMPLATE.md](./TEMPLATE.md) como base. La spec se escribe antes de implementar, y se actualiza si el alcance cambia durante el desarrollo.
